#简单三子棋 (Simple Three-in-a-Row)

#这是一个很简单的三子棋游戏（井字棋盘）！不仅可以和电脑下，还可以和好朋友一起下。最酷的是，你可以把背景和棋子都换成你喜欢的图片！

#这个游戏在经典的3x3棋盘上进行，但加入了有趣的移动机制：

#1.  **放棋子阶段**：你和对手轮流放棋子,每人手手里只有3个棋子,先把这 3 个棋子都放到棋盘上。
#2.  **挪棋子阶段**：当你们的 3 个棋子都放完后，就不能再拿新棋子了,轮到你时，点击一个你的棋子（它会变红），然后把它移动到**旁边相邻**的空位上,**目标**：谁先让自己的 3 个棋子连成一条线（横着、竖着、斜着都行）,谁就赢了
#支持更换棋盘和棋子图片更换
